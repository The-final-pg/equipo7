# GIT FLOW

## pasos

**crear rama**

- git branch name
- git cheackout -b name (crea la rama y te para en ella)

**borar rama**

- git branch -d name
- shift + ctrl + p y escribir: >rama o branch

**traer ramas nuevas**

- git fetch

**actualizar las que se borraron**

- git remote update origin --prune
